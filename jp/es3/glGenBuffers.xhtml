Name

glGenBuffers — バッファーオブジェクトの名前を生成する

仕様

void glGenBuffers( GLsizei n,
  GLuint * buffers);
 

パラメーター

n

生成するバッファオブジェクト名を指定する

buffers

生成するバッファオブジェクト名が格納される配列を指定する

説明

glGenBuffersはbuffersのバッファオブジェクト名をnに返す。 連続した整数から名前を返す保証はない； しかし、glGenBuffersを呼び出す直前に返された名前が未使用だったことは保証される。

glGenBuffersの呼び出しによって返されるバッファオブジェクト名は、glDeleteBuffersによって削除されない限り、後の呼び出しで返されない。

buffersに返された名前はglGenBuffersのみの目的で使用済みとしてマークされるが、 glBindBufferの呼び出しによって最初にバウンドされるときにのみ、ステートとタイプを取得する。

エラー

GL_INVALID_VALUE is generated if n is negative. nが負の数のときにGL_INVALID_VALUEが発生する

関連する項目

glIsBuffer
